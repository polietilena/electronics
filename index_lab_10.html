Дослідження тригерів

 

Ціль роботи

Ціллю роботи є  дослідження асинхронних і синхронних тригерів в статичному і динамічному режимах.

 

Методичні вказівки

 

По літературі і конспекту лекцій вивчити наступні питання:
–  вивчити  умовно-графічні позначень тригерів;
–  вивчити принцип побудови і логіки роботи  тригерів на інтегральних  мікросхемах.

 

Короткі теоретичні відомості

 

Тригер –  це запам’ятовуючий елемент з двома стійкими станами, які змінюються під впливом вхідних сигналів. Як елемент ЕОМ, тригер призначений для зберігання біта інформації.

 

Тригери класифікують по наступним ознакам:


– логіці  функціонування (RS, JK, D, T і ін.);
– способу  запису інформації (асинхронні, синхронні);
– моменту  реакції за тактовим сигналам (статичні, динамічні);
– кількості тактів синхронізації (однотактні, двотактні);
– кількості ступенів  (одно- або двуступеневі тригери).


За способом запису інформації розрізняють асинхронні і синхронні тригери. Тригери, що не мають управляючого (синхронізуючого) С-входа, називаються асинхронними. Умовно-графічне позначення (УГП) асинхронних тригерів приведене на рис. 1.



Рис.1. Умовно-графічне позначення  асинхронних тригерів: а) RS; б)  JK; в) Т

 

У асинхронних тригерах запис інформації проводиться у будь-який момент часу під час поступлення сигналів на інформаційні входи. Тригери, що мають управляючий С-вхід, називається синхронними. На С-вхід поступають періодичні імпульси, які називаються синхронізуючими або тактовими. УГП синхронних тригерів приведене на рис.2.

Рис. 3. Вимірювання тимчасових параметрів тригерів

Двоступеневий JK-тригер на елементах І-НІ

Завдання і порядок виконання роботи

 

Дослідження асинхронного RS-тригера.


1. Відкрийте файл лаб_10а.ewb (схема віртуальної лабораторної установки зображена на  рис.12) для перевірки закону функціонування асинхронного RS-тригера.



Рис.12. Схема віртуальної лабораторної установки для перевірки  закону функціонування асинхронного RS-тригера.

 

Схема установки включає перемикачі SA1 (клавіша ”1”), SA2 (клавіша ”2”) для задавання зміни значень вхідних змінних R і S;  індикаторів рівнів для визначення значень вхідних змінних R і S і станів  вихідних змінних;  вольтметрів V1 і V2  для  виміру вихідної  напруги, що відповідає  логічним ”1” чи ”0”.


2. Підготувати віртуальну установку до роботи:
– перемикачі  SA1 і SA2 за допомогою клавіш “1”, ”2” встановити в нижнє положення, при якому вхідні змінні R і S рівні 0.


3.  Включити установку на моделювання. При цьому тригер переключається в нульовий стан: на Q виході “0”, а на Q' -  “1”. Це відповідає нульовому набору таблиці істинності. За допомогою вольтметрів  V1 і V2 виміряти значення рівнів “1” і “0”, і в таблиці істинності замість логічних нулів і одиниць для виходів Q і  Q' записати виміряне значення напруги.


4.  Змінюючи положення перемикачів SA1 і SA2 перевірити всі набори таблиці істинності. Вимкнути моделювання. Заповнити таблицю переходів  підставивши замість нулів і одиниць відповідні покази вольтметрів.


5. Відкрийте файл лаб_10б.ewb (схема віртуальної лабораторної установки зображена на        рис. 13) для дослідження RS-тригера в динамічному режимі.

 


Рис.13. Схема віртуальної лабораторної установки для дослідження RS-тригера в динамічному режимі

 

В схему входять: асинхронний RS-тригер; генератор слів, логічний аналізатор.


6. Двічі клацнути лівою кнопкою миші по піктограмі генератора слів.  На генераторі слів необхідно перевірити або установить наступні режими:


1. Step, для покрокового виконання;


2.Internal, для внутрішнього запуску;
3. Запуск по передньому фронту клавішею .
4. Frequency: 150Hz
У лівому вікні генератора слова перевірити наступні|слідуючі| шість наборів:
0 0 0 A
0 0 0 9
0 0 0 С  
0 0 0 2
0 0 0 4
0 0 0 9.

 

7. Включити схему на моделювання.  При цьому на логічному аналізаторі тими ж кольорами, що і провідники входів S, R і виходів Q,  Q' , будуть по тактах малюватися їх логічні рівні. Після 5-6 тактів слід вимкнути режим моделювання.

 

8. Перемальовувати в протокол зображення з логічного аналізатора і скласти таблицю переходів тригера по графіку аналізатора.
